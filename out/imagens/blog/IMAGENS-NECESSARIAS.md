# Blog Images

Este diretório contém as imagens dos posts do blog.

## Especificações

- Formato: JPG, PNG ou WEBP
- Dimensões: 1200x630 pixels (recomendado)
- Tamanho: Menor que 500KB para melhor performance
- Tema: Aviação, aeronaves, ferry flight

## Como Adicionar

1. Coloque as imagens neste diretório (`/public/imagens/blog/`)
2. Nomeie os arquivos conforme listado acima
3. Ou faça upload através do painel admin em `/admin/blog`
